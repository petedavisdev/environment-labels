## Development
First, run

```
npm install
```

Workflow:

1. The following command builds the extension and watches for changes to js files.

```
npm run dev
```

2. Open Chrome
3. Go to Manage Extensions
4. Enable developer mode
5. Load unpacked
6. Choose the `build` directory
