Settings

Must be the correct JSON format

{
  "style": "font-size: 16px; padding: 8px;",
  "markers": [
    {
      "name": "Local",
      "include": [
        "localhost:",
        "local.yourdomain.com"
      ],
      "color": "green"
    },
    {
      "name": "Live!",
      "include": "https://yourdomain.com",
      "color": "#ffcc33"
    }
  ]
}
